# land

A desktop-optimized, Zillow-style prototype for homesteaders evaluating land by livability metrics (not developer flip potential).

## Features

- Listing search + filtering (acres, max price, slope, water rights, and well availability).
- Detailed property insights for:
  - Price, acres, parcel number.
  - Average slope and percent of land under 10° (Google Earth style terrain fields).
  - Water source, power source, well/septic presence, water rights.
  - Geology notes.
  - Annual rainfall for food-growth planning.
  - Distances to Costco, hospital, town, hardware store, and airport.
  - Tax history summary.
- Add/edit style workflow through a data entry form that updates the UI immediately.
- Homestead viability score to aggregate terrain, water infrastructure, rainfall, and access.

## Run locally

Open `index.html` in a modern desktop browser.
