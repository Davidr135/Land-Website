# Homestead Finder concept

A lightweight front-end concept for a Zillow-style marketplace focused on land buyers who want off-grid or semi-off-grid living.

## What is included

- A polished landing page that explains the product concept.
- Interactive parcel filters for budget, acreage, water access, and buildable land.
- Persona-based ranking for balanced, off-grid, family-ready, and food-production buyers.
- Featured parcel cards showing the exact kinds of land-specific fields the platform should highlight.
- A local services section for providers like well drillers, solar installers, builders, and site prep teams.

## Run locally

Because this is a static prototype, you can open `index.html` directly in a browser or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
